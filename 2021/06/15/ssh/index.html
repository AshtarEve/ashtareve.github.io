linux 配置 ssh port 以及免密登录

linux 配置 ssh port 以及免密登录

  1. 打开配置文件（需要 root 权限）

  2. 配置相关属性

    属性名称 值 备注
    Port 数字 端口号
    RSAAuthentication yes RSA 密钥认证
    PubkeyAuthentication yes 公钥认证
    PasswordAuthentication no 密码认证

  3. 保存配置文件，重启ssh服务

  4. 添加本地计算机公钥

  5. 使用本地私钥进行登录

拓展

  1. vim 命令
    vim 文件名，打开文件
    输入 「i」 进入编辑模式，可以进行编辑内容
    输入 「Esc」 推出编辑模式
    输入 「:」 进入命令模式，输入相应命令后回车执行各种操作

    命令 作用
    wq 保存退出
    q! 不保存退出

  2. 本地生成 rsa 密钥对: ssh-keygen 命令
    输入自定义路径，以及密钥对相关信息即可
    id_rsa 私钥文件
    id_rsa.pub 公钥文件

  3. ssh -T git@github.com
    在 github 配置公钥后，可以使用该命令验证是否成功
