使用 Hexo 和 Icarus 搭建博客

Hexo 项目地址： https://github.com/hexojs/hexo
Icarus 项目地址： https://github.com/ppoffice/hexo-theme-icarus

Hexo

A fast, simple & powerful blog framework, powered by Node.js.

Hexo 是基于 Node.js 的一个博客框架，搭建 Hexo 博客需要先安装 Node.js。

Icarus

A simple, delicate, and modern theme for the static site generator Hexo.

Icarus 是 Hexo 的主题之一，用于美化博客。

开始搭建

  1. 进入 Node.js 官方网站, 下载最新的 LTS (长期支持) 版本安装包。

  2. 安装 Node.js 之后，使用 npm 下载安装 hexo 脚手架。

  3. 使用 hexo 脚手架创建博客项目。

  4. 进入 blog 文件夹，下载 icarus 主题到 hexo 的 themes 文件夹下。

  5. 设置 hexo 的主题为 icarus。

  6. 安装 hexo 依赖。

  7. 启动 hexo server。

    2021 年 5 月 20 日，hexo (version 5.4.0), icarus (version 4.2.0) 版本，需要手动再安装上述提示的依赖。（icarus 引用的依赖未被成功安装）

    由于找不到 hexo-component-inferno@0.11.0 ，仍然会安装失败。可以使用 yarn 命令选择可用的版本安装，或者前往 hexo-component-inferno 项目，查看并使用最新 release 的版本进行安装。成功安装 icarus 依赖后再执行一次 npm install 去除循环依赖的包。

    再次执行 hexo server 输入如下结果。使用浏览器访问 http://localhost:4000/ 查看博客效果。

第一篇博文

在 blog 文件夹下，执行 hexo new "my first blog" 会在 source\posts 下生成 my-first-blog.md 文件，编写该文件保存再执行hexo server 即可浏览最新博文。

Tips： 可以使用 hexo s 简写命令直接启动 server。

更多命令以及指南可以参看官方文档：hexo 官方文档， Icarus 快速上手。
