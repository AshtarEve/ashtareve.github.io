2021-05-24еҸ‘иЎЁ2021-05-24жӣҙж–°node3 еҲҶй’ҹиҜ»е®Ң (еӨ§зәҰ377дёӘеӯ—)

е…ідәҺ node.js зҡ„зүҲжң¬

жң¬ж–ҮеҸӮиҖғ node.js е®ҳзҪ‘еҸ‘еёғйЎөйқўпјҢе№¶ж·»еҠ дёҖдәӣи§ЈйҮҠгҖӮ

еҸ‘еёғ

дё»зүҲжң¬зҡ„ Node.js иҝӣе…Ҙ еҪ“еүҚзүҲ(Current) е°ҶжҢҒз»ӯе…ӯдёӘжңҲзҡ„ж—¶й—ҙпјҢеңЁжӯӨжңҹй—ҙеә“дҪңиҖ…еҸҜд»ҘеҜ№е…¶иҝӣиЎҢж”ҜжҢҒгҖӮ е…ӯдёӘжңҲд№ӢеҗҺпјҢеҘҮж•°зүҲжң¬пјҲиҜёеҰӮ 9гҖҒ11 зӯүпјүе°ҶеҸҳдёәдёҚж”ҜжҢҒзҠ¶жҖҒпјҢеҸӘжңүеҒ¶ж•°зүҲжң¬пјҲиҜёеҰӮ 10гҖҒ12 зӯүпјүеҸҳжҲҗ жҙ»и·ғ LTS (Active LTS) зҠ¶жҖҒпјҢе№¶дё”еҮҶеӨҮжҠ•е…ҘдҪҝз”ЁгҖӮ LTS еҸ‘еёғзүҲзҡ„зҠ¶жҖҒжҳҜвҖңй•ҝжңҹж”ҜжҢҒзүҲ(long-term support)вҖқпјҢиҝҷж„Ҹе‘ізқҖйҮҚеӨ§зҡ„ Bug е°ҶеңЁеҗҺз»ӯзҡ„ 30 дёӘжңҲеҶ…жҢҒз»ӯеҫ—еҲ°дёҚж–ӯең°дҝ®еӨҚгҖӮ дёҠзәҝеә”иҜҘд»…дҪҝз”Ё жҙ»и·ғ LTS (Active LTS) жҲ–иҖ…жҳҜ з»ҙжҠӨ LTS (Maintenance LTS) зүҲгҖӮ

йҖҡеёёпјҢnode.js жҜҸе№ҙ 4 жңҲд»ҪеҸ‘еёғеҒ¶ж•°зүҲжң¬пјҢ10 жңҲд»ҪеҸ‘еёғеҘҮж•°зүҲжң¬гҖӮ

жҰӮеҝө

  1. Current: еҪ“еүҚжңҖж–°зҡ„зүҲжң¬пјҢиҜҘйҳ¶ж®өйҖҡеёёжҢҒз»ӯе…ӯдёӘжңҲзҡ„ж—¶й—ҙпјҢеә“дҪңиҖ…еҸҜд»ҘеҜ№е…¶иҝӣиЎҢж”ҜжҢҒгҖӮ
  2. Active LTSпјҡжҢҮжӯЈеңЁз§ҜжһҒз»ҙжҠӨе’ҢеҚҮзә§зҡ„зүҲжң¬зі»еҲ—пјҢеҢ…жӢ¬еҗ‘еҗҺз§»жӨҚйқһз ҙеқҸжҖ§еҠҹиғҪе’Ңж”№иҝӣпјҢи§ЈеҶій”ҷиҜҜд»ҘеҸҠдҝ®иЎҘе®үе…ЁжјҸжҙһгҖӮ
  3. Maintenance LTSпјҡжҢҮз»ҙжҠӨзҡ„ LTS зүҲжң¬пјҢжӣҙеҠ зЁіе®ҡпјҢзӣҙеҲ°е®ғзҡ„з”ҹе‘Ҫе‘Ёжңҹз»ҲжӯўпјҢеҸӘдјҡеңЁзҹӯж—¶й—ҙеҶ…ж”¶еҲ°й”ҷиҜҜдҝ®еӨҚе’Ңе®үе…ЁиЎҘдёҒгҖӮ
  4. End-of-lifeпјҡиҝӣе…ҘеҲ° EOL ж—¶й—ҙзәҝзҡ„зүҲжң¬пјҢе°ҶдёҚеңЁз»ҙжҠӨгҖӮ

Hexo

Icarus

A simple, delicate, and modern theme for the static site generator Hexo.

Icarus жҳҜ Hexo зҡ„дё»йўҳд№ӢдёҖпјҢз”ЁдәҺзҫҺеҢ–еҚҡе®ўгҖӮ

  2. е®үиЈ… Node.js д№ӢеҗҺпјҢдҪҝз”Ё npm дёӢиҪҪе®үиЈ… hexo и„ҡжүӢжһ¶гҖӮ

  3. дҪҝз”Ё hexo и„ҡжүӢжһ¶еҲӣе»әеҚҡе®ўйЎ№зӣ®гҖӮ

  4. иҝӣе…Ҙ blog ж–Үд»¶еӨ№пјҢдёӢиҪҪ icarus дё»йўҳеҲ° hexo зҡ„ themes ж–Үд»¶еӨ№дёӢгҖӮ

  5. и®ҫзҪ® hexo зҡ„дё»йўҳдёә icarusгҖӮ

    з”ұдәҺжүҫдёҚеҲ° hexo-component-inferno@0.11.0 пјҢд»Қз„¶дјҡе®үиЈ…еӨұиҙҘгҖӮеҸҜд»ҘдҪҝз”Ё yarn е‘Ҫд»ӨйҖүжӢ©еҸҜз”Ёзҡ„зүҲжң¬е®үиЈ…пјҢжҲ–иҖ…еүҚеҫҖ hexo-component-inferno йЎ№зӣ®пјҢжҹҘзңӢе№¶дҪҝз”ЁжңҖж–° release зҡ„зүҲжң¬иҝӣиЎҢе®үиЈ…гҖӮжҲҗеҠҹе®үиЈ… icarus дҫқиө–еҗҺеҶҚжү§иЎҢдёҖж¬Ў npm install еҺ»йҷӨеҫӘзҺҜдҫқиө–зҡ„еҢ…гҖӮ

    еҶҚж¬Ўжү§иЎҢ hexo server иҫ“е…ҘеҰӮдёӢз»“жһңгҖӮдҪҝз”ЁжөҸи§ҲеҷЁи®ҝй—® http://localhost:4000/ жҹҘзңӢеҚҡе®ўж•ҲжһңгҖӮ
